# 🎓 Student Performance Analyzer

A Python desktop application that helps manage student records, analyze academic performance, and visualize statistics using graphs.

---

## 📌 Features

- Add student details
- Calculate average marks automatically
- Update existing student records
- Display class statistics
- Visualize student performance using bar charts
- Input validation with error messages
- Object-Oriented Programming (OOP) design
- Clean and modular project structure

---

## 🛠️ Technologies Used

- Python 3
- Tkinter (GUI)
- NumPy
- Matplotlib
- Object-Oriented Programming (OOP)
- Git & GitHub

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/HimaniNimma/Student_Performance_Analyzer_Project.git
```

Move into the project:

```bash
cd Student_Performance_Analyzer_Project
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python main.py
```
